# Guide : 2 joueurs sur le même PC avec manettes

## C'est quoi le mode 2 joueurs ?

**Les deux joueurs jouent sur le MÊME PC**, devant le même écran. Pas besoin de réseau ni d'un deuxième ordinateur.

- **1 PC** + **1 écran** + **2 manettes** (ou 1 manette + 1 clavier)
- L'écran est **partagé** : en Joueur vs Joueur, moitié gauche = J1, moitié droite = J2
- En 2 Joueurs vs Bot, les deux joueurs sont visibles ensemble dans le même niveau

---

## Préparation

### 1. Brancher les manettes **avant** de lancer le jeu

- Connectez les **deux manettes PS4** en USB (ou Bluetooth si configurées) **avant** d'ouvrir le jeu.

### 2. Activer les manettes dans le jeu

1. Lancez **Cyber Jump**
2. Menu principal → **Paramètres**
3. **Manette PS4** : réglez sur **Activée**
4. Retour au menu

### 3. Choisir un mode à 2 joueurs

- **Menu** → **Mode Compétitif**
- Choisissez :
  - **Joueur vs Joueur** : même niveau pour les deux (écran partagé)
  - **2 Joueurs vs Bot** : les deux dans le même niveau contre le bot

### 4. Choisir vos contrôles (écran de sélection)

Quand vous choisissez **Joueur vs Joueur** ou **2 Joueurs vs Bot**, un écran s'affiche. **Chaque joueur choisit** avec sa manette ou le clavier :

- **Bouton X** sur VOTRE manette → Vous êtes Joueur 1
- **Bouton Cercle** sur VOTRE manette → Vous êtes Joueur 2
- **Espace** (clavier) → Joueur 1 au clavier
- **Entrée** (clavier) → Joueur 2 au clavier
- **P** ou **Options** (manette) → Passer (J1=Manette1, J2=Manette2)

---

## Contrôles complets

### Avec 2 manettes

| Action | Joueur 1 | Joueur 2 |
|--------|----------|----------|
| **Course** (déplacer) | Stick gauche ← → | Stick gauche ← → |
| **Course** (sauter) | Bouton X (carré) | Bouton X (carré) |
| **Combat** (attaquer) | Bouton X | Bouton X |

### Avec clavier uniquement

| Action | Joueur 1 | Joueur 2 |
|--------|----------|----------|
| **Course** (gauche) | Flèche ← ou A | J |
| **Course** (droite) | Flèche → ou D | L |
| **Course** (sauter) | Espace | I |
| **Combat** (attaquer) | Espace | Entrée |

### Mixte (1 manette + 1 clavier)

| | Joueur 1 | Joueur 2 |
|--|----------|----------|
| **Manette** | Stick + Bouton X | — |
| **Clavier** | — | J / L / I (course) + Entrée (combat) |

Ou inversement : J1 au clavier (Flèches + Espace), J2 à la manette (Stick + Bouton X).

---

## Dépannage

| Problème | Solution |
|----------|----------|
| La 2ᵉ manette ne répond pas | Vérifiez qu’elle est branchée **avant** le lancement du jeu |
| Aucune manette ne fonctionne | Paramètres → Manette PS4 : **Activée** |
| Joueurs inversés | Débranchez les deux manettes, rebranchez dans l’ordre souhaité, puis relancez le jeu |
| Manette non reconnue | Testez sur un autre port USB ou réinstallez les pilotes |

---

## Résumé rapide

1. Brancher les 2 manettes **avant** de lancer le jeu  
2. Paramètres → Manette PS4 : **Activée**  
3. Mode Compétitif → Joueur vs Joueur ou 2 Joueurs vs Bot  
4. **Écran de sélection** : X = Joueur 1, Cercle = Joueur 2 (ou Espace/Entrée pour clavier)  
5. Jouer avec le stick gauche et le bouton X
